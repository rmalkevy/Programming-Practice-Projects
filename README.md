# Software Design Practice Projects

A collection of practical programming labs for students who have completed the first year of computer science or software engineering studies.

## Labs

1. Mini Messenger
2. Wolfenstein-like Ray Casting Engine
3. Simple Ray Tracer
4. STM32 Sensor Logger / Simulator
5. Image Processor
6. Digital Tree Visualizer
7. Graph Route Finder
8. Fractal Generator
9. Console Paddle Game
10. Maze Generator and Solver
11. Mini File Explorer
12. Task Tracker
13. Physics Sandbox
14. Cellular Automata Simulator
15. Mini Search Engine
